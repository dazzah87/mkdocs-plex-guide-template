# Was ist Plex?

Plex ist eine Medien-Streaming-Plattform, die Deinen Computer in einen persönlichen Netflix-ähnlichen Dienst verwandelt. Wenn Plex auf einem Server installiert ist, schafft sie eine Benutzeroberfläche, über die Du Filme und Fernsehsendungen von jedem Gerät aus ansehen kannst – egal ob zu Hause oder unterwegs. Wenn Du Dich mit meinem Plex-Server verbindest, erhältst Du Zugriff auf meine Medienbibliothek, ohne selbst einen Server einrichten oder warten zu müssen.

Offizielle Webseite: [plex.tv](https://www.plex.tv/)
